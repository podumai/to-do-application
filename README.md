# to-do-application

A lightweight, high-performance desktop application build with **C++23** and **Qt Quick**.  
This project demonstrates the integration of a fluid, hardware-accelerated QML interface with a robust **SQLite** backend for persistent note management.

# Tech Stack

| Component        | Technology |
|------------------| --- |
| **Language**     | C++23 |
| **Framework**    | Qt 6.x (Qt Quick / QML) |
| **Database**     | SQLite 3 |
| **Build System** | CMake |
| **Architecture** | Model-View-Controller |
